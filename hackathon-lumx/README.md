## ğŸ›  Installation (Mobile)

1. **Pre-requisites**
    - Make sure you have Dart and Flutter installed on your machine.

2. **Clone the Repository**

    ```bash
    git clone https://github.com/bellujrb/hackathon-lumx/frontend
    ```

3. **Install Dependencies**

    ```bash
    flutter pub get
    ```

4. **Run the App**

    ```bash
    flutter run
    ```
